# Social People App
This app emit to make people have online interaction. it provides enabling them from its and make them social.
Person can:
- Make post and all friends view its, 
- Be chat with another person or group Chat,
- he receives notification on any message.
- Add, upload Stories and See how many views it has.
- add a live broadcast while allowing followers to enter and exit it and add 
comments to it .
- receives notifications on the post or when any message is received.
- Profile contains (Stories – Posts – Number of Friends – Following – Photos 
that do upload at post)
